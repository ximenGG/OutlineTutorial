# IOS客服端
IOS客服端只能到appstore下载,由于在中国区appstore没有此款App,所以要去香港区,台湾,美国区的appstore下载此App.

# 关注微信公众号OpenInternet
###  回复appleid-usa获取美国地区的AppleId
###  回复appleid-tw获取台湾地区的AppleId

# 连接
Outline支持SS和SSR协议,扫描SS或者SSR的二维码即可.
